# CONFIGURATION ############################################################### 

tiles.ini ---------------------------------------------------------------------
Tiles for the map and their properties.


settings.ini ------------------------------------------------------------------
All of the general settings for this package.


themes.ini --------------------------------------------------------------------


# MODULES #####################################################################


canvas.py ---------------------------------------------------------------------
Management of cartesian coordinate canvas system.


brush.py ----------------------------------------------------------------------
The brush to "paint" the canvas!

Typically tools for defining area sets.


example.py --------------------------------------------------------------------
Example code of how one might use this package.


mapgen.py ---------------------------------------------------------------------
Map generator, which primarily uses cartmap.py. Is the abstraction layer for
"map stuff," e.g., rooms, doors, paths, etc.


display.py --------------------------------------------------------------------
Presentation for the user.


database.py -------------------------------------------------------------------
Management of SQLITE3 database for this package.
